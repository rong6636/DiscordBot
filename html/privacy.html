DiscordBot 隱私權政策

一、資料收集說明

本 Discord 機器人（以下簡稱「本服務」）在提供功能過程中，可能會接觸到以下資訊：

  • 用戶的 Discord 使用者名稱與 ID
  • 伺服器 ID（Guild ID）與頻道名稱
  • 使用者透過指令主動提供的查詢參數

本服務不會主動儲存或分析訊息內容，也不會將資訊提供給第三方。

二、資料用途

所有資料僅用於即時回應使用者指令，並不會持久儲存或用於其他用途。

三、第三方服務

本服務可能會使用第三方開放資料 API，例如中央氣象局資料開放平台、新聞 RSS 資料等，所有查詢皆匿名執行，與使用者身份無關。

四、資料安全

本服務部署於雲端平台，僅在記憶體中暫存資料處理，不進行資料庫長期儲存。

五、使用者權利

如您對個人資料使用有疑慮，或希望我們停止特定互動，您可透過移除機器人或聯絡開發者以撤除使用。

六、政策修改

本隱私權政策可能隨機器人功能調整而更新。最新版本將公開發布於本頁面。

七、聯絡方式

如有任何問題，請至 GitHub 倉庫 https://github.com/rong6636/DiscordBot 提出 issue 聯繫開發者。
